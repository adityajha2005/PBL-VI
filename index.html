Enhancing ML Performance on Small Datasets

Applied to Forecasting Urban Air Pollution Spikes in Jaipur

A small-data-oriented framework combining domain-knowledge-driven feature engineering with virtual sample generation (SMOTE-for-regression) to improve PM₂.₅ and PM₁₀ forecasting performance for Tier-2 cities with limited monitoring infrastructure.

View Full Paper

Problem Statement

Air quality forecasting in Tier-2 cities like Jaipur suffers from limited monitoring infrastructure and small datasets. Extracting meaningful predictions, especially during extreme pollution events (e.g., Diwali festival), is challenging with standard ML approaches. This project addresses: (1) scarcity of high-quality air quality data, (2) class imbalance for rare extreme-pollution events, (3) lack of domain-specific features for Indian festivals that cause pollution spikes.

Literature Review / Market Research

Existing air quality forecasting work focuses on metropolitan cities with larger datasets. Traditional approaches (ARIMA, Prophet, basic ML) struggle with small datasets and rare-event prediction. SMOTE is well-established for classification; its adaptation for regression with domain-specific features in Indian urban contexts remains underexplored.

Research Gap / Innovation

First framework combining domain-knowledge features (Diwali ±21d, Lohri ±7d with continuous days_since_festival), SMOTE-for-regression to augment extreme-pollution samples (PM₂.₅ > 150 µg/m³), and a three-feature strategy (temporal, domain, autoregressive) designed for Indian urban context.

System Methodology

Dataset / Input

Police Commissionerate, Jaipur (Site 134, RSPCB) — 8-hourly measurements 2019–2025, 7,671 observations. Features: PM2.5, PM10, NO, NO₂, NH₃, SO₂, CO, Ozone. Train: 5,247 (2019–2023) | Test: 2,423 (2024–2025).

Model / Architecture

Random Forest (150 trees, max_depth=10). Feature engineering: 14 features (temporal sin/cos, domain Diwali/Lohri, lag-1 & lag-2). SMOTE augmentation for extreme events (PM₂.₅ > 150 µg/m³).

Results & Analysis

PM₂.₅ R² (Enhanced vs Baseline) 50.3% vs 45.5%
PM₂.₅ MAE (µg/m³) 13.09 vs 13.87

SMOTE-enhanced Model B outperforms the baseline (Model A) by better capturing rare Diwali pollution spikes. Diwali-period R² peaks at 0.855 (2022); test years 2024–2025 show improved MAE with augmentation. PM₁₀ predictions: MAE 30.53 vs 30.82 (baseline); R² 0.460 vs 0.457.
